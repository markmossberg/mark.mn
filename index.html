Projects

librequests

A simple C library for making HTTP requests. It actually wraps around libcurl (so I guess it's a level two library?) Written to help me learn C.

lockpick

A script I wrote to let me unlock my dorm door via the command line. It piggybacks off of a little known web interface provided by Northeastern that lets a student access functions such as changing their ID card PIN, and unlocking their door from a mobile website. After doing some elementary reverse engineering of the web application, I wrote this script which automates the process of logging a user into the web interface, and triggering the door unlock mechanism. For more info, read my blog post.

valas

Valas stands for the "Vengeful Auth.Log Analysis Script" and was semi-joke project that I worked on at Hack Beanpot 2014. It's a script that processes an Ubuntu server's auth.log file and analyzes the invalid login attempts, producing a chart of the countries, and some other analytics. It also port scans every invalid host and lets you know what services they're running.

carve

This was my hack for the 2013 Cipher Tech Mobile Forensics Hackathon. It won 1st place.

The challenge at this hackathon was to extract the most information from a supplied iPhone system image. I was able to extract a good amount of the data during the hackathon, but I've continued to work on thisto improve the script and extract more data.

tuneSniffer

A script I wrote that extracts mp3 files from sniffed network traffic. Written in order to improve my knowledge of TCP/IP networking and Python.

airport+

Macs come with a pretty awesome wifi stumbling utility called airport that lets you see all sorts of cool information about nearby wireless networks, including SSID, BSSID, Channel, and encryption method. One feature I would have liked also was the vendor of the router, so I decided to create a wrapper script that implemented this feature.
