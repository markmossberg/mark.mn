Projects

carve

This was my hack for the 2013 Cipher Tech Mobile Forensics Hackathon.

The challenge at this hackathon was to extract the most information from a supplied iPhone system image. I was able to extract a good amount of the data during the hackathon, but I've continued to work on this to improve the script and extract more data.

tuneSniffer

A script I wrote that extracts mp3 files from sniffed network traffic. Written in order to improve my knowledge of TCP/IP networking and Python.

airport+

Macs come with a pretty awesome wifi stumbling utility called airport that lets you see all sorts of cool information about nearby wireless networks, including SSID, BSSID, Channel, and encryption method. One feature I would have liked also was the vendor of the router, so I decided to create a wrapper script that implemented this feature.

randomcamp

A simple utility that allows people to find new music through almost-random artists on bandcamp.com. Created to learn how to use APIs.
